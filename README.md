# callerID - тестовый проект предоставление данных по номеру телефона.

API метод /api/caller-id/<phone> принимает номер абонента в формате
MSISDN (например, 79173453223), возвращает информацию о
номере (операторе мобильной связи и регионе), в случае ошибки —
сигнализировать об этой ошибке. Формат JSON.

В корневом url предоставлена web форма принимает номер телефона, а в ответ
показывает информацию о номере (операторе мобильной связи и регионе).

Периодическая задача в полночь загружает реестры российской системы и плана
нумерации https://opendata.digital.gov.ru/registry/numeric/downloads, заносит в БД.
